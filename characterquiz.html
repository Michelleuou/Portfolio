Character Quiz

Description

This is a project where I create a website of a character quiz. I used Delatrune, a game to create my character quiz. I used 4 character, Kris, Susie, Berdly, and Ralsei. While creating my character quiz, I'm suppose to use conditional statements to show which character the user has gotten after submitting the quiz.

Goal

The goal was to use conditiona statements in JQuery and understanding conditional statements in my code.

Challenges

  • After all conditions weren't met in the code, the dog should have been displayed. However, the dog kept appearing along with the other character, when condition was met.
  • The conditional statements were very disorganized and sepreated on it own block

Solution

I figured that the dog should be as shown by default to the code, as well as putting the dog into a else statment. This way the dog is able to be displayed when the conditions aren't met and the dog doesn't overlap other character's when condition is met. I combined each character into one block and put them in a else if statement. This way the code is more readable. Also this took a big chunk of unnecesscary code in my if statement.
